## ü§ñ MCP Server (AI IDE Integration)

Omnitok UI Kit cuenta con un servidor **MCP (Model Context Protocol)** que permite a editores con IA consultar la documentaci√≥n y especificaciones de los componentes directamente desde el IDE.

**Endpoint**: `https://ui-kit.omnitok.cloud/mcp`

### Configurar en Cursor

Agrega la siguiente entrada en `.cursor/mcp.json` (a nivel de proyecto o global):

```json

    "omnitok-ui-kit": {

    }
  }
}
```

### Configurar en VS Code (Copilot)

Agrega la entrada en `.vscode/mcp.json`:

```json
{
  "servers": {
    "omnitok-ui-kit": {
      "type": "sse",
      "url": "https://ui-kit.omnitok.cloud/mcp"
    }
  }
}
```

### Configurar en Windsurf

Agrega la entrada en `~/.codeium/windsurf/mcp_config.json`:

```json
{
  "mcpServers": {
    "omnitok-ui-kit": {
      "serverUrl": "https://ui-kit.omnitok.cloud/mcp"
    }
  }
}
```
