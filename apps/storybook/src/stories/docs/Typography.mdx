# Tipografía

Sistema tipográfico para interfaces administrativas.

## Familias Tipográficas

### Primary (UI)
**Nunito Sans** - Para interfaz de usuario, botones, navegación

```css
font-family: 'Nunito Sans', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
```

**Clase Tailwind**: `font-sans`

<Typeset
  fontSizes={[Number(typography.size.base), Number(typography.size.xl), Number(typography.size['2xl'])]}
  fontWeight={typography.weight.regular}
  sampleText={SampleText}
  fontFamily={typography.type.primary}
/>

**Características**:
- Humanista sans-serif
- Alta legibilidad en pantalla
- Soporta múltiples pesos
- Excelente para UI elements

### Secondary (Contenido)
**Open Sans** - Para contenido, tablas, texto largo

```css
font-family: 'Open Sans', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
```

**Clase Tailwind**: `font-mono`

<Typeset
  fontSizes={[Number(typography.size.base), Number(typography.size.xl), Number(typography.size['2xl'])]}
  fontWeight={typography.weight.regular}
  sampleText={SampleText}
  fontFamily={typography.type.secondary}
/>

**Características**:
- Neutral y profesional
- Optimizada para lectura
- Excelente en tablas y datos

## Buenas Prácticas

1. **Jerarquía clara**: Usar tamaños y pesos para establecer jerarquía visual
2. **Consistencia**: Mantener la misma familia para elementos similares
3. **Legibilidad**: Line-height mínimo de 1.5 para texto de contenido
4. **Contraste**: Asegurar contraste suficiente con el fondo (mínimo 4.5:1)
5. **Responsive**: Reducir tamaños en móvil si es necesario
