## Accesibilidad

- **Contraste mínimo**: 4.5:1 para texto normal, 3:1 para texto grande
- **No depender solo del color**: Usar íconos, texto o patrones adicionales
- **Texto sobre fondos de color**: Siempre usar blanco (#FFFFFF) sobre primary/accent/sidebar
