# Sistema de Colores

Paleta de colores para sistemas administrativos internos basada en la identidad Omnitok.

## Colores de Marca

### Sidebar
- **Sidebar Background**: `#211f4b` - Color principal del sidebar izquierdo
- **Uso**: Fondo del sidebar de navegación (obligatorio en todos los proyectos)

### Primary
- **Primary**: `#4D4A9D` - Púrpura principal
- **Primary Dark**: `#3B3880` - Variante oscura para hover/active
- **Primary Light**: `rgba(77, 74, 157, 0.1)` - Variante clara para fondos

**Uso**: Botones primarios, enlaces, elementos interactivos principales, headers de tablas/cards/modals.

### Accent
- **Accent**: `#FF177B` - Magenta/Rosa vibrante
- **Accent Dark**: `#E0156E` - Variante oscura
- **Accent Light**: `rgba(255, 23, 123, 0.1)` - Variante clara

**Uso**: CTAs importantes, badges de notificación, elementos que requieren máxima atención. Usar con moderación.

### Gradientes

#### Elemento Activo en Sidebar
```css
background: linear-gradient(90deg, #FF177B 0%, #4D4A9D 100%);
border-radius: 8px;
margin: 2px 8px;
color: #ffffff;
font-weight: 500;
```

#### Headers (Cards, Modals, Tables)
```css
background: linear-gradient(90deg, #4D4A9D 0%, #3B3880 100%);
color: #FFFFFF;
```

#### Tabs Pills (Activo)
```css
background: linear-gradient(90deg, #FF177B 0%, #4D4A9D 100%);
color: #FFFFFF;
```

## Colores Semánticos

### Success
- **Success**: `#10B981` - Verde para estados exitosos
- **Success Light**: `rgba(16, 185, 129, 0.1)` - Fondo de alertas success
- **Success Dark**: `#059669` - Hover/active

### Warning
- **Warning**: `#F59E0B` - Amarillo/naranja para advertencias
- **Warning Light**: `rgba(245, 158, 11, 0.1)` - Fondo de alertas warning
- **Warning Dark**: `#D97706` - Hover/active

### Error
- **Error**: `#EF4444` - Rojo para errores y acciones destructivas
- **Error Light**: `rgba(239, 68, 68, 0.1)` - Fondo de alertas error
- **Error Dark**: `#DC2626` - Hover/active

### Info
- **Info**: `#3B82F6` - Azul para información
- **Info Light**: `rgba(59, 130, 246, 0.1)` - Fondo de alertas info
- **Info Dark**: `#2563EB` - Hover/active

### Metabase
Usar la misma paleta de colores para consistencia visual.

## Accesibilidad

- **Contraste mínimo**: 4.5:1 para texto normal, 3:1 para texto grande
- **No depender solo del color**: Usar íconos, texto o patrones adicionales
- **Texto sobre fondos de color**: Siempre usar blanco (#FFFFFF) sobre primary/accent/sidebar
